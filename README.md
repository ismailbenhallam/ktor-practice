# ktor-practice

### This project demonstrate the key features provided by Ktor server

### In this project you could learn how to:

- Use some (of the main) plugins:
    - Routing
    - Resources
    - Authentication
    - ContentNegotiation
    - StatusPages
    - CallLogging
    - ShutDownUrl
- Handle events
- Test your server
- Extend Ktor:
    - Create & use custom plugins
    - Create custom events
- Create a fat JAR
